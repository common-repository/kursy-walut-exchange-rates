== Description ==

PL:
Wtyczka uruchamia usługę poprzez umożliwienie dodania do swojego bloga tabelki z kursami walut za pomocą widgetu lub shortcodu. Dostępne pary walutowe to: EUR/PLN, USD/PLN, CHF/PLN oraz GBP/PLN.

Kursy dostarcza https://ekantor.pl

EN:
The plugin adds a service and table of exchange rates to your blog using the widget or shortcod. Courses are updated by using server ekantor.pl. Available currency pairs are: EUR/PLN, USD/PLN, CHF/PLN and GBP/PLN.

Courses provides https://ekantor.pl

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place a widget or shortcode [kw_currency_table] on your blog
